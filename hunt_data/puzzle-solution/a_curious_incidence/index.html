Solution to A Curious INCIDENCE

Author: Alina Griner

All of the clues are paraphrased quotes from Through the Looking Glass. The punctuation is kept the same but words/phrases in the sentence are replaced by synonymous words/phrases. Each clue is missing one word from the original phrase. The first step is to find the missing words.

The second step is to notice that all of the missing words, if written in capital letters just as the clues, fit the same pattern: every letter in the word has the same symmetry type except one.

Symmetry types (letters in multiple categories in parentheses):

BCDEK(HIOX) − reflectional symmetry about the horizontal axis
AMTUVWY(HIOX) − reflectional symmetry about the vertical axis
NSZ(HIOX) − reflectional symmetry about the origin
FGJLPQR − no symmetry

For each group, taking the odd letter out from each missing word and reading these letters in order spells another word. Repeating this process with the new words, one for each group, spells out "UTENSIL". This is the answer to the puzzle.
