Solution to Crosswalks

Authors: Chieu Nguyen and Aaron Bader

This puzzle is a list of intersections in the city of Chicago, which has streets mostly laid out on a grid oriented with the cardinal directions. This grid is centered at State St and Madison St, which is the origin of the coordinate system, and all streets are named with a directional prefix (N/S, E/W) with respect to this origin. Furthermore, the addressing system for the entire city is based on this grid, so that any street address gives one of the two coordinates for that point on the grid. Every increase in 800 roughly corresponds to a mile in distance.

In this puzzle, each pair of coordinates indicates an intersection of two streets. In keeping with the theme of intersections, you solve the puzzle by treating each street name (not including the directional prefix or the street type (St, Rd, Ave, etc.)) as a set of letters and taking the intersection of the two sets. For all of the given street intersections, the corresponding letter intersection is a singleton set. Order these letters from north to south, and the letters spell out BULLS POINT GUARD ROSE, which clues the answer DERRICK, in reference to point guard Derrick Rose of the Chicago Bulls.
